记一次右键菜单失效修复注册表的过程

问题

右键文件，点击Open with Sublime Text失效，提示弹窗：该文件没有与之关联的应用来执行该操作。请安装应用，若已经安装应用，请在“默认应用设置”页面中创建关联。

处理过程

  1. 迷茫
    首先 win+R ——> regedit 打开注册表。
    然后在导航栏输入：计算机\HKEY_CLASSES_ROOT\Applications\sublime_text.exe\shell\open\command，进去后发现是有命令的：”E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe” “%1”，那么为什么右键会失效呢？

  2. 找到问题
    在网上搜索后，找到了原因。
    在导航栏中输入：计算机\HKEY_CLASSES_ROOT\/*\shell\Open with Sublime Text，进去后发现该项下没有相应的command，于是在Open with Sublime Text下新建一个项command，编辑command的值为：”E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe” “%1”，解决！

扩展

1）在右键菜单栏处新增图表

定位到：计算机\HKEY_CLASSES_ROOT\/*\shell\Open with Sublime Text，选中Open with Sublime Text，新建一个字符串值，命名为Icon，其值为sublime_text.exe的路径（本机是”E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe”）

2）增加右键文件夹，用Sublime打开功能

  1. 设置 右键单击文件夹时在右键菜单中显示
    1）定位到：计算机\HKEY_CLASSES_ROOT\Directory\shell，在shell项下新增一项，命名为OpenWithSublimeText，对OpenWithSublimeText下的(默认)值设置值为Open Folder with Sublime Text 3，然后在OpenWithSublimeText下新增一个字符串值，命名为Icon，其值为E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe。
    2）然后，在OpenWithSublimeText下新建一项，命名为command，设置其值为”E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe” “%1”

  2. 设置 右键单击资源管理器的空白区域时在右键菜单中显示
    1）定位到：计算机\HKEY_CLASSES_ROOT\Directory\Background\shell，在shell项下新增一项，命名为Sublime Text3，对Sublime Text3下的(默认)值设置值为Open Folder with Sublime Text 3，然后在Sublime Text3下新增一个字符串值，命名为Icon，其值为E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe。
    2）然后，在Sublime Text3下新建一项，命名为command，设置其值为”E:\Program Files (x86)\sublime\Sublime Text 3\sublime_text.exe” “%1”
