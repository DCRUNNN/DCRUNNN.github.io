windows下查看端口被占用并结束进程

Jan 5, 2019 | 技术整理 Windows

步骤

  1. win+R cmd 启动命令行
  2. netstat -ano | findstr “端口号”，回车就可以看到这个端口被哪个进程占用 （记住最后一列的值）
  3. tasklist | findstr “进程id号”，查看到对应的进程id之后，就可以通过id查找对应的进程名称，使用命令
  4. taskkill /f /t /im “进程id或者进程名称”，通过命令杀掉进程，或者是直接根据进程的名称杀掉所有的进程
